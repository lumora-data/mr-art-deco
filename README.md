# Mr. Light Art Deco

Site Next.js App Router pour Mr. Light Art Deco, spécialiste en électricité bâtiment, rénovation complète, décoration intérieure et installation de luminaires à Douala.

## Développement

```bash
npm install
npm run dev
```

## Déploiement Vercel

- Framework preset : `Next.js`
- Root directory : racine du dépôt
- Build command : `npm run build`
- Output directory : laisser la valeur par défaut de Next.js

Les images locales sont servies depuis `public/images` et référencées avec des chemins absolus comme `/images/logo.jpeg`.
